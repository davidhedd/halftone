# halftone - photo app for professional photographers #

Description:
	Photography mobile app based on geotagging images/photos, aimed specifically towards professional photographers.
	It allows users to search for snaps based on geographic location and filter based on attributes such tags and date/time.

Features(Implemented):
	User login
	User registration
	Take photo
	Like/Unlike other users photos
	Display a users liked photos(list of ids)

Features(Future Releases):
	Choose image from gallery
	Tag photo location
	Show images based on a geographical location input
	Filtering based on tags or location
	Customise photo location
	Display photo data(username, location, date taken, tags...)
	Display likes in editable view(view / unlike), with image thumbnail and basic details
	Swipe between content screens
	Swipe like/unlike photos
	Search for specific user / tags...

Android technologies / features used:
	Firebase -> Authentication / Real time database
	Activities
	Fragments
	General android layout/design principles

Android technologies / features researched, not yet implemented:
	Google places API
	GSON
	Common gestures(swipe)
